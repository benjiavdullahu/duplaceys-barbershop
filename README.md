# Duplacye's Barbershop Website

A modern website for Duplacye's Barbershop in downtown Hamilton.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Built With

- Next.js 15
- Tailwind CSS
- TypeScript
